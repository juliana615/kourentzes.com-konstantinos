== Description ==

An animated, easy-to-use, feature-rich, Lightbox Block popup plugin for any web content such as images, audio, video, and much more.

You can easily create image gallery, audio gallery, video gallery and content gallery using lightbox block.

**[Demos](https://bplugins.com/products/lightbox-block/#demos)**


= Features =

- First Select LightBox Type.
- There are two types of Gallery and Button.
- If you Select LightBox type gallery then multiple items are added.
- LightBox Type Button select then go to Style Panel and style change in button.
- Upload your images, audio, video, and content to the plugin.
- Supports YouTube, Vimeo and Video.
- The LightBox will automatically engage with galleries/photos.
- Mobile responsive so it looks awesome on all devices.
- Use different modes like slideshow or FullScreen.
- Receive dedicated support for your questions.
- Completely customizable.
- Two type layout-Default, Masonry.

= How to use =
- First install the Lightbox block plugin
- Add the Lightbox block from the block category called "Widgets" in Gutenberg editor.
- You can change block settings from the right side settings sidebar.
- Enjoy!

* For installation help click on Installation Tab

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Lightbox block**
3. Click on the **Lightbox block** to add the block

= Download & Upload: =
1. Download the **Lightbox block** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`lightbox-block.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **Lightbox-block** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
